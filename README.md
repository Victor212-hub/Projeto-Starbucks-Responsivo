#  Projeto Starbucks Landing Page

# Sobre o projeto

Este é um projeto de uma landing page inspirada no site da Starbucks, desenvolvido com foco em prática de **HTML, CSS e JavaScript básico**.

O objetivo principal foi treinar:

* Estruturação de páginas com HTML
* Estilização com CSS (Flexbox, animações e transições)
* Manipulação do DOM com JavaScript

---

# Funcionalidades

* Troca dinâmica da imagem do produto
* Alteração da cor de fundo conforme o copo selecionado
* Animações ao passar o mouse (hover effects)
* Layout responsivo básico

---

# Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

# Layout

O projeto apresenta:

* Uma seção principal com destaque para o produto
* Texto promocional
* Botões interativos
* Miniaturas (copos) que alteram o conteúdo da tela

# Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos como:

* Manipulação de elementos com `querySelector`
* Alteração de atributos (`src`)
* Mudança dinâmica de estilos com JavaScript
* Uso de `flexbox` para organização de layout
